Política de Privacidade

Última atualização: 2026-03-09

Resumo: A aplicação Ponto Zero funciona em modo offline-first. Os dados são guardados no teu dispositivo. A app não cria contas, não recolhe dados pessoais e não envia dados financeiros para servidores.

2. Princípios fundamentais da aplicação

A aplicação Ponto Zero foi desenvolvida segundo os princípios de:

  • Privacidade por design
  • Minimização de dados
  • Armazenamento local

A aplicação:

  • Não cria contas de utilizador
  • Não requer login
  • Não solicita nome, email ou contactos
  • Não recolhe localização
  • Não identifica diretamente o utilizador
  • Não possui servidores próprios para armazenamento de dados financeiros
  • Não envia dados financeiros para a internet
  • Não integra publicidade
  • Não integra sistemas de analytics
  • Não integra rastreadores
  • Não partilha dados com terceiros

A aplicação funciona em modelo offline-first, sendo possível utilizar as suas funcionalidades principais sem ligação à internet.

3. Dados armazenados

Todos os dados introduzidos pelo utilizador são armazenados exclusivamente no dispositivo.

Exemplos de dados guardados localmente:

  • Registos de receitas
  • Registos de despesas
  • Contas financeiras
  • Categorias
  • Movimentos de poupança
  • Empréstimos e respetiva evolução
  • Preferências da aplicação
  • Definições de segurança (PIN e opções de biometria)

O desenvolvedor não tem acesso a estes dados.

4. Segurança e autenticação

PIN

A aplicação pode permitir a proteção através de PIN.

O PIN é armazenado localmente utilizando mecanismos de armazenamento seguro disponibilizados pelo sistema operativo do dispositivo.

Biometria

A autenticação biométrica (por exemplo impressão digital ou reconhecimento facial):

  • É processada exclusivamente pelo sistema operativo do dispositivo
  • Não é recolhida pela aplicação
  • Não é armazenada pela aplicação

A aplicação apenas recebe uma confirmação de sucesso ou falha da autenticação.

5. Ligações à internet

A aplicação não necessita de ligação à internet para funcionar nas suas funcionalidades principais.

Nenhum dado financeiro introduzido pelo utilizador é transmitido online pelo desenvolvedor.

A aplicação pode abrir ligações externas, por exemplo para:

  • website oficial
  • páginas de suporte

Ao aceder a serviços externos, passam a aplicar-se as respetivas políticas de privacidade desses serviços.

6. Backups e exportação de dados

A aplicação permite ao utilizador efetuar backups dos seus dados.

Backups manuais

  • O ficheiro é criado localmente
  • A exportação ocorre apenas quando solicitada pelo utilizador
  • O utilizador decide onde guardar ou partilhar o ficheiro
  • Após a exportação, o controlo do ficheiro passa a ser exclusivamente do utilizador

Backups automáticos (Premium)

  • Na versão Premium, o utilizador pode ativar backups automáticos locais
  • Os backups automáticos são gerados no próprio dispositivo
  • Nenhum ficheiro é enviado para servidores do desenvolvedor
  • O utilizador é responsável pela gestão e armazenamento dos ficheiros gerados

O desenvolvedor não recebe nem armazena cópias de backups, sejam manuais ou automáticos.

7. Permissões do dispositivo

A aplicação pode solicitar permissões apenas quando necessárias para determinadas funcionalidades, como:

  • Acesso ao armazenamento (para exportação ou importação de backups)
  • Utilização de biometria (quando ativada pelo utilizador)

Nenhuma permissão é utilizada para recolha de dados pessoais, rastreamento ou criação de perfis de utilizador.

O sistema operativo do dispositivo pode disponibilizar informações técnicas mínimas necessárias ao funcionamento dessas permissões.

8. Retenção dos dados

Os dados permanecem armazenados no dispositivo até que o utilizador:

  • Apague manualmente os registos
  • Limpe os dados da aplicação
  • Desinstale a aplicação

O desenvolvedor não mantém qualquer cópia externa desses dados.

9. Direitos do utilizador (RGPD)

Como os dados permanecem apenas no dispositivo do utilizador, os direitos previstos no RGPD podem ser exercidos diretamente na aplicação:

  • Direito de acesso — o utilizador pode consultar os dados armazenados na aplicação
  • Direito de retificação — o utilizador pode editar ou corrigir os seus registos
  • Direito ao apagamento — o utilizador pode eliminar dados ou desinstalar a aplicação
  • Direito à portabilidade — o utilizador pode exportar os seus dados através da funcionalidade de backup

10. Crianças

A aplicação não recolhe dados pessoais.

Não é dirigida especificamente a menores de 16 anos, podendo ser utilizada sob supervisão responsável.

11. Funcionalidades Premium

Algumas funcionalidades podem estar disponíveis mediante pagamento.

  • As compras são processadas exclusivamente pela Google Play ou Apple App Store
  • O desenvolvedor não tem acesso a dados de pagamento
  • O desenvolvedor não recebe dados pessoais associados à transação
  • O desenvolvedor não gere reembolsos
  • Os reembolsos seguem exclusivamente as políticas da respetiva loja

A aquisição de funcionalidades Premium não implica recolha adicional de dados pessoais.

12. Limitação importante

O Ponto Zero é uma ferramenta de organização financeira pessoal.

A aplicação não fornece:

  • aconselhamento financeiro
  • aconselhamento fiscal
  • aconselhamento contabilístico
  • recomendações de investimento

Todas as decisões financeiras tomadas com base na aplicação são da responsabilidade do utilizador.

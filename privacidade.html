Política de Privacidade

Última atualização: 01 de Março de 2026

Resumo: o Ponto Zero funciona em modo offline-first. Os dados são guardados no teu dispositivo. A app não cria contas, não recolhe dados pessoais e não envia dados financeiros para servidores.

1. Responsável pelo tratamento

Ricardo Santos — Portugal

Email: suporte.app.pontozero@gmail.com

2. Princípio fundamental da aplicação

O Ponto Zero foi desenvolvido segundo os princípios de:

  • Privacidade por design
  • Minimização de dados
  • Armazenamento local

A aplicação:

  • Não cria contas
  • Não requer login
  • Não solicita nome, email ou contactos
  • Não recolhe localização
  • Não identifica o utilizador
  • Não possui servidores próprios
  • Não envia dados financeiros para a internet
  • Não integra publicidade
  • Não integra analytics
  • Não integra rastreadores
  • Não partilha dados com terceiros

A aplicação funciona de forma offline-first, sendo totalmente utilizável sem ligação à internet.

3. Dados armazenados

Todos os dados introduzidos pelo utilizador são armazenados exclusivamente no dispositivo.

Exemplos de dados guardados localmente:

  • Registos de receitas
  • Registos de despesas
  • Contas financeiras
  • Categorias
  • Movimentos de poupança
  • Empréstimos e respetiva evolução
  • Preferências da aplicação
  • Definições de segurança (PIN e biometria)

O desenvolvedor não tem acesso a estes dados.

4. Segurança e autenticação

PIN: o PIN é armazenado localmente utilizando mecanismos de armazenamento seguro disponibilizados pelo sistema operativo do dispositivo.

Biometria: a autenticação biométrica (impressão digital ou reconhecimento facial):

  • É processada exclusivamente pelo sistema operativo do dispositivo.
  • A aplicação não recebe nem armazena dados biométricos.
  • Apenas recebe confirmação de sucesso ou falha da autenticação.

5. Ligações à internet

A aplicação não necessita de ligação à internet para funcionar.

Nenhum dado financeiro é transmitido online.

A aplicação pode abrir ligações externas (por exemplo, para o website oficial). Ao sair da aplicação, aplicam-se as políticas de privacidade do respetivo website.

6. Backups e exportação de dados

A aplicação permite ao utilizador efetuar backups dos seus dados.

Backups manuais:

  • O ficheiro é criado localmente.
  • A exportação ocorre apenas quando solicitada.
  • O utilizador decide onde guardar ou partilhar o ficheiro.
  • Após a exportação, o controlo do ficheiro passa a ser exclusivamente do utilizador.

Backups automáticos (Premium):

  • Na versão Premium, o utilizador pode ativar backups automáticos locais.
  • Os backups automáticos são gerados no próprio dispositivo.
  • Nenhum ficheiro é enviado para servidores do desenvolvedor.
  • O utilizador continua a ser responsável pela gestão e armazenamento dos ficheiros gerados.

7. Permissões do dispositivo

A aplicação pode solicitar permissões apenas para:

  • Acesso a armazenamento (para exportação/importação de backups)
  • Utilização de biometria (se ativada pelo utilizador)

Nenhuma permissão é utilizada para recolha de dados pessoais ou rastreamento.

8. Retenção dos dados

Os dados permanecem no dispositivo até que o utilizador:

  • Apague manualmente os registos
  • Limpe os dados da aplicação
  • Desinstale a aplicação

O desenvolvedor não mantém qualquer cópia externa dos dados.

9. Direitos do utilizador (RGPD)

Como os dados não são transmitidos ao desenvolvedor, os direitos podem ser exercidos diretamente no dispositivo:

  • Direito de acesso — consultar dados na aplicação
  • Direito de retificação — editar registos
  • Direito ao apagamento — eliminar dados ou desinstalar
  • Direito à portabilidade — exportar backup

10. Crianças

A aplicação não recolhe dados pessoais.

Não é dirigida especificamente a menores de 16 anos, podendo ser utilizada sob supervisão responsável.

11. Funcionalidades Premium

Algumas funcionalidades podem estar disponíveis mediante pagamento.

  • As compras são processadas pela Google Play ou Apple App Store.
  • O desenvolvedor não tem acesso a dados de pagamento.
  • O desenvolvedor não gere reembolsos.
  • Reembolsos seguem exclusivamente as políticas da respetiva loja.

A aquisição de funcionalidades Premium não implica recolha adicional de dados pessoais.

12. Limitação importante

O Ponto Zero é uma ferramenta de organização financeira pessoal.

Não fornece:

  • aconselhamento financeiro
  • aconselhamento fiscal
  • aconselhamento contabilístico
  • recomendações de investimento

Todas as decisões financeiras tomadas com base na aplicação são da responsabilidade do utilizador.

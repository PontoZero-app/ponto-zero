Suporte

Ajuda e suporte

Se precisares de ajuda com a aplicação Ponto Zero, consulta as soluções abaixo. Muitas situações comuns podem ser resolvidas rapidamente.

Problemas comuns

Perdi os meus dados

Os dados são guardados apenas no dispositivo.

Se os dados desapareceram, as causas mais comuns são:

  • desinstalação da aplicação
  • limpeza de dados nas definições do sistema
  • substituição ou formatação do dispositivo

Se tiveres um backup, podes restaurar os dados. Sem backup, infelizmente não é possível recuperar a informação.

Troquei de telemóvel

Para transferir os dados para um novo dispositivo é necessário ter um backup.

Passos recomendados:

  1. Criar backup no dispositivo antigo
  2. Copiar o ficheiro para o novo dispositivo
  3. Restaurar o backup na aplicação

Sem backup, os dados não podem ser recuperados.

Esqueci o PIN

Por motivos de segurança, o PIN não pode ser recuperado.

Se não conseguires desbloquear a aplicação, será necessário:

  • limpar os dados da aplicação nas definições do sistema
  • ou reinstalar a aplicação

Atenção: isto irá apagar os dados guardados localmente.

A aplicação não abre ou fecha sozinha

Tenta os seguintes passos:

  1. Reiniciar o dispositivo
  2. Atualizar a aplicação
  3. Verificar espaço de armazenamento disponível
  4. Atualizar o sistema operativo

Se o problema continuar, contacta o suporte.

A biometria não funciona

Verifica se:

  • o dispositivo suporta biometria
  • tens impressão digital ou reconhecimento facial configurado
  • o PIN está ativo na aplicação
A aplicação está lenta

Isto pode acontecer quando o dispositivo tem pouca memória disponível ou muitas aplicações abertas em simultâneo.

Tenta:

  • fechar outras aplicações
  • reiniciar o dispositivo
  • confirmar se a aplicação está atualizada
Não consigo restaurar um backup

Verifica se:

  • o ficheiro pertence à aplicação Ponto Zero
  • o ficheiro não foi alterado manualmente
  • o ficheiro está completo e não foi corrompido

Se o ficheiro estiver incompleto ou danificado, o restauro pode não ser possível.

Não encontro o ficheiro de backup

Os ficheiros de backup são guardados no dispositivo no momento da exportação.

Dependendo do dispositivo, podem aparecer em:

  • pasta de transferências
  • armazenamento interno
  • localização escolhida pelo utilizador durante a exportação

Backups

Como proteger melhor os meus dados?

Criar backups regularmente é a melhor forma de proteger os teus dados.

Se mudares de telemóvel, perderes o dispositivo ou apagares a aplicação, o backup pode ser a única forma de recuperar a informação.

Backup manual

Podes exportar um ficheiro de backup nas definições da aplicação.

Esse ficheiro pode ser guardado em:

  • armazenamento do dispositivo
  • computador
  • serviços de cloud
Backup automático (Premium)

Na versão Premium podes ativar backup automático local.

Os backups são criados no próprio dispositivo. Nenhum dado é enviado para servidores do desenvolvedor.

Compras Premium

Comprei Premium e as funcionalidades não aparecem

Tenta os seguintes passos:

  1. Verificar se estás ligado à mesma conta da loja
  2. Reiniciar a aplicação
  3. Confirmar se a aplicação está atualizada

Se o problema continuar, contacta o suporte.

Como funcionam reembolsos?

As compras são processadas pela Google Play ou Apple App Store.

Os reembolsos seguem as políticas da respetiva loja.

Perguntas rápidas

A aplicação usa internet?

Não. A aplicação funciona principalmente offline.

Os meus dados são enviados para servidores?

Não. Todos os dados são armazenados apenas no dispositivo.

A aplicação liga à minha conta bancária?

Não. O Ponto Zero não se liga a bancos nem acede a contas bancárias.

Posso usar a aplicação em vários dispositivos?

A aplicação não sincroniza dados entre dispositivos.

Para transferir dados entre dispositivos é necessário utilizar backup e restauro.

Funcionalidades

O que podes fazer com o Ponto Zero

O Ponto Zero foi desenvolvido para fornecer controlo financeiro claro e previsível. Cada funcionalidade existe para responder a uma necessidade real de organização financeira.

Registo de receitas e despesas

Permite registar todos os movimentos financeiros manualmente.

Cada registo inclui:

  • valor
  • categoria
  • conta
  • data
  • descrição opcional

A separação entre despesas fixas e variáveis permite compreender o peso das obrigações mensais.

Categorias personalizáveis

O utilizador pode criar, editar ou arquivar categorias.

Exemplos:

  • alimentação
  • transporte
  • subscrições
  • seguros
  • lazer

As categorias são usadas para gerar resumos e análise de gastos.

Gestão de contas

A app suporta múltiplas contas:

  • contas bancárias
  • carteira física
  • cartão de crédito
  • contas poupança

O saldo apresentado reflete o valor real após todos os movimentos.

Resumos financeiros

Os resumos mostram a situação financeira consolidada.

Disponível por:

  • mês
  • semana (Premium)
  • ano (Premium)
  • período personalizado (Premium)

Inclui:

  • total de receitas
  • total de despesas
  • saldo
  • evolução

Património

Permite acompanhar poupanças e evolução financeira ao longo do tempo.

Inclui:

  • movimentos de poupança
  • transferências internas
  • ajustes de valor

O objetivo é mostrar crescimento financeiro real.

Gestão de empréstimos

O utilizador pode registar empréstimos e acompanhar a sua evolução.

Inclui:

  • valor em dívida
  • prestações restantes
  • histórico de pagamentos

Sempre que uma prestação é registada, o valor em dívida é atualizado automaticamente.

Exportação e backup

Os dados podem ser exportados manualmente para ficheiro.

Funções disponíveis:

  • exportar backup
  • restaurar backup
  • backup automático (Premium)

Todos os dados permanecem locais no dispositivo.

Segurança

A app inclui proteção por:

  • PIN
  • autenticação biométrica (Premium)

O acesso pode ser bloqueado automaticamente após inatividade.

Personalização

O utilizador pode personalizar:

  • tema claro/escuro
  • ícones principais (Premium)

Funcionamento offline

A aplicação funciona totalmente sem ligação à internet.

Nenhum dado é enviado para servidores externos.

Modelo de utilização

O Ponto Zero não automatiza decisões financeiras.

A aplicação apresenta informação organizada para que o utilizador tome decisões conscientes.

Não fornece aconselhamento financeiro nem recomenda investimentos.

1. Finalidade da aplicação

O Ponto Zero é uma ferramenta de organização financeira pessoal, que pode incluir funcionalidades como:

  • Registo de receitas e despesas
  • Gestão de contas, património, poupanças e dívidas/empréstimos
  • Resumos e indicadores de organização financeira
  • Exportação/backup de dados

A Aplicação não está ligada a instituições bancárias, nem acede a contas bancárias do Utilizador.

2. Ausência de aconselhamento

A Aplicação não fornece aconselhamento financeiro, fiscal, contabilístico ou recomendações de investimento. Qualquer informação, resumo ou indicador apresentado é de natureza meramente organizacional.

O Utilizador é responsável por:

  • Efetuar backups caso pretenda conservar/restaurar dados

4. Armazenamento local, backups e perda de dados

Os dados são armazenados localmente no dispositivo do Utilizador. A Aplicação não mantém, por defeito, servidores próprios com dados financeiros do Utilizador.

Não é possível garantir a recuperação de dados apagados.
